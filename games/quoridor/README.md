# DinoBoard Intelligence - Quoridor 模块

`games/quoridor/` 是 Quoridor（双人）在 DinoBoard 通用框架中的实现。

当前实现包含：

- C++ 规则引擎（走子、跳跃/斜跳、放墙与通路校验）
- C++ selfplay / arena 接口
- 调试服务（开局、合法动作、人类落子、AI 落子、悔棋）
- 训练入口（`train/plugin.py`）与 Torch 简化训练器

## 规则范围

- 9x9 棋盘，双方初始在中线（`(1,5)` 与 `(9,5)`）
- 每名玩家 10 面墙
- 每回合可“走子”或“放墙”
- 走子支持相邻跳跃；若背后被墙或边界阻挡，允许斜跳
- 放墙后必须保证双方仍至少有一条到达目标线的路径

## 网络输入说明

- NetMCTS 使用当前执子方视角编码特征（对手方会做 180 度翻转）。
- 若特征编码规则调整，旧模型将与新编码不兼容，需要重新训练。
